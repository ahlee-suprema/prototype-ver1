# Suprema T&A System - UI Mockup

이 디렉토리는 Suprema Time & Attendance 시스템의 프론트엔드 목업을 포함합니다.

## 목업 파일 목록

| 파일 | 설명 | 주요 기능 |
|------|------|----------|
| `index.html` | **메인 대시보드** | 직원/HR 관리자 대시보드, 출퇴근 현황, 휴가 현황 |
| `leave-request.html` | **휴가 신청** | 휴가 유형 선택, 기간 선택, 결재선 확인 |
| `overtime-request.html` | **연장근무 신청** | 사전/사후 신청, 주간 연장근무 현황, 12시간 제한 |
| `attendance-records.html` | **출퇴근 기록** | 일별 기록 조회, 월별 통계, 엑셀 다운로드 |
| `team-management.html` | **팀 현황 (관리자)** | 팀원 근무 현황, 승인 대기, 52시간 모니터링 |
